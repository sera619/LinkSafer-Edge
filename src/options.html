Link Safer is a browser extension for Microsoft Edge and Google Chrome that allows users to save links and notes securely. Its been planned a feature to generate secure passwords. Link Safer is created by S3R43o3 using HTML, CSS, and JavaScript. The extension uses the storage and tabs permissions to store and access the user's data. The extension has an options page where users can customize their settings, and a popup where users can view and manage their links, notes, and passwords. Link Safer is a useful and convenient tool for anyone who wants to keep their online data safe and organized.
