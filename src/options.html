Link Safer is a browser extension for Microsoft Edge and Google Chrome that allows users to save links and notes securely. It has a planned feature to generate secure passwords. Link Safer is created by S3R43o3 using HTML, CSS, and JavaScript. The extension uses the storage and tabs permissions to store and access the user's data. The extension has an options page where users can customize their settings and a popup where users can view and manage their links, notes, and passwords. Link Safer is a useful and convenient tool for anyone who wants to keep their online data safe and organized.




User instructions



  1. To add a new link or notice, enter the URL or text in the input field and click the "Add" button.
  2. To add the current active tab to the list, click the "Add Current Tab" button.
  3. To remove a link or notice from the list, click the "X" button next to it.
  4. To open a link, click on the link text.
  5. To add a link to bookmarks, click the "+" button next to it.
  6. To clear the entire list, click the "Clear List" button.
  7. To download the list as a text file, click the "Download" button.
  8. To open the options/help menu, click the "Options" button.
  9. To open the password generator, click the "Password Generator" button.
  10. To save the current tab session, click the "Save Session" button.
  11. To restore the saved tab session, click the "Restore Session" button.

Please note that URLs starting with "http", "https", "chrome:", "edge:", or "chrome-extension:" are considered safe and clickable.
